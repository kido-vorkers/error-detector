エラーをみやすくするやつ
====================

某アレのエラーを見やすくします。
Mac版Google Chromeで動作を確認しましたが理論上はSafari, Opera, Chromium, Iron, ComodoDragonなどでも動くと思います。

Install
-------

1. Google Chromeの設定で拡張機能の設定を開き、デベロッパーモードにチェックを入れます
2. 「パッケージ化されていない拡張機能を読み込む」で`manifest.json`のあるディレクトリを指定します
